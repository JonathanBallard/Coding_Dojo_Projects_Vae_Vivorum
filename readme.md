
# Vae Vivorum

**Twin Stick Shooter  
Based on the pre-bootcamp fighter jet game**  

# All Features

## Features - MVP

- Create Player Ship  
- Create Enemies Ships  
- Movement  
- Fire Weapons  
- Enemies  
- Destroy Enemies  
- Take Hits from Enemies  
- Win After certain number of enemies dead  
- XP gain  
- Score gain  
- Stored player info in database
